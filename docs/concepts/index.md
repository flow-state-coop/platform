# Concepts

This section explains the ideas behind **Flow State** and the vocabulary we use across the platform. If the [Platform](../platform/flow-councils/index.md) pages tell you *how* to use the tools, these pages tell you *why* they work the way they do.

Flow State is a streaming money platform built on [Superfluid](https://docs.superfluid.finance/docs/concepts/superfluid). Instead of moving money in lump sums, we move it as continuous, real-time **streams** — so capital and results can stay in sync.
