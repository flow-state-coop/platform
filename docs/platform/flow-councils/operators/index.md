# Running a Round

Everything a round operator does happens in the [launchpad](https://flowstate.network/flow-councils/launch). This section walks through the flow step by step, in the order you'll meet it:

1. **[Launch](001-launch.md)** — deploy the council and pick the network and distribution Super Token.
2. **[Permissions](002-permissions.md)** — grant the **Super Admin**, **Voter Review**, and **Recipient Review** roles.
3. **[Applications & Review](003-applications.md)** — build the application form with the no-code form builder and review submissions.
4. **[Council Membership](004-membership.md)** — define who votes, organize voter groups, and set vote budgets.
5. **[Funding](005-funding.md)** — sponsor the funding stream, manage the SuperApp Splitter, and schedule the round end.
6. **[Communications](006-communications.md)** — broadcast announcements and chat with projects.

Looking for the voter, applicant, or funder side? See [Participating](../participants/index.md).
