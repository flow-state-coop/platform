# Account

Your Flow State account ties together everything tied to your wallet — your public identity, the messages you receive, and how you're notified.

- **[Profile](001-profile.md)** — set your display name, bio, and social links, and add a private email for applications.
- **[Inbox](002-inbox.md)** — a unified inbox for application updates, project channels, and round announcements.
- **[Email Notifications](003-notifications.md)** — choose which emails Flow State sends you, and unsubscribe at any time.
